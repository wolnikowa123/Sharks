
# Sharks Travel — strona szkółki sportowej (React + TypeScript + styled-components)

## Szybki start
```bash
npm install
npm run dev
```
Wejdz na adres z konsoli (domyślnie: http://localhost:5173).

## Kolory
Paleta znajduje się w `src/theme.ts`. Możesz zmienić kody HEX, jeśli chcesz idealnie dopasować je do brandu z grafiki.

## Edycja treści
- Harmonogramy: `src/data/schedules.ts`
- Sekcje: komponent `src/components/Schedule.tsx`
- Cennik: `src/components/Pricing.tsx`
- Trenerzy: `src/components/Coaches.tsx`
- Opinie: `src/components/Testimonials.tsx`
- Kontakt / mail: `src/components/Contact.tsx`
- Zdjęcia i loga: wrzuć do `src/assets/` i podmień placeholdery.

## Build produkcyjny
```bash
npm run build
npm run preview
```
